# ideas.alanwsmith.com 

This is the source code for
[ideas.alanwsmith.com](https://ideas.alanwsmith.com/)

## Updates

I keep the source item list in my Grimoire in the 
file: 

> site- Ideas For Programming Projects For The Subdomain.neo

New items are added there then I regenerate the list
by running:

> content_builder/build_content.py

which outputs them to:

> componements/IdeasData.js

## Next Steps
